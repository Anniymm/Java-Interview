#  1️⃣ Harshad Number
An **Harshad Number** is an integer that is divisible by the **sum of its digits**.
## 📘 Definition
- In base 10:  
  `n % sum_of_digits(n) == 0`
- **n-Harshad Number**:  
  A number divisible by the sum of its digits in **base n**.
## 📌 Examples
- 18 → 1 + 8 = 9 → 18 % 9 = 0 ✅  
- 19 → 1 + 9 = 10 → 19 % 10 ≠ 0 ❌
